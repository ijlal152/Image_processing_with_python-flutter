# Rename Screenshots Before Pushing to GitHub

Please rename your screenshots manually to match the README:

## In Finder:
1. Open `/screenshots/` folder
2. Rename the files as follows:

```
Screenshot 2026-04-16 at 3.56.34 PM.png  →  webapp-home.png
Screenshot 2026-04-16 at 3.57.32 PM.png  →  webapp-result-auto.png
Screenshot 2026-04-16 at 3.58.16 PM.png  →  webapp-result-denoise.png
Screenshot 2026-04-16 at 4.01.29 PM.png  →  mobile-home.png
Screenshot 2026-04-16 at 4.06.20 PM.png  →  mobile-image-selected.png
Screenshot 2026-04-16 at 4.07.05 PM.png  →  mobile-result.png
```

## OR use this command:
```bash
cd screenshots
mv "Screenshot 2026-04-16 at 3.56.34 PM.png" webapp-home.png
mv "Screenshot 2026-04-16 at 3.57.32 PM.png" webapp-result-auto.png
mv "Screenshot 2026-04-16 at 3.58.16 PM.png" webapp-result-denoise.png
mv "Screenshot 2026-04-16 at 4.01.29 PM.png" mobile-home.png
mv "Screenshot 2026-04-16 at 4.06.20 PM.png" mobile-image-selected.png
mv "Screenshot 2026-04-16 at 4.07.05 PM.png" mobile-result.png
```

After renaming, delete this file!
